# Athletaker

Athletaker is a web application designed for **Manchester United fans** to track their favorite players, view fixtures and results, and explore the club's historical achievements. Built using **HTML, CSS, JavaScript, and PHP**, this project provides an interactive and informative experience for fans.

---

## Features

- **Player Tracker**: View detailed information about Manchester United players, including stats and profiles.  
- **Fixtures & Results**: Stay updated with upcoming matches and recent results.  
- **Historical Achievements**: Explore the club’s rich history and past achievements.  
- **Responsive Design**: Works smoothly across desktops, tablets, and mobile devices.
